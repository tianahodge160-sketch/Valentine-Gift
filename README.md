Valentine's Day page

Open `Valentine Gift.html` in a browser to view the page.

Quick steps (Windows):

1. Double-click `Valentine Gift.html` in the `valentine-site` folder.
2. Or open a terminal and run:

```powershell
start .\valentine-site\Valentine Gift.html
```

Customize the message by typing a name and message, then click "Reveal My Message". Click "Make it Rain Hearts" for a bigger effect.
